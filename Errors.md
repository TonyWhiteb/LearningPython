# __try...except...finally__

```python
try:
    print('try...')
    r = 10 / 0
    print('result:', r)
except ZeroDivisionError as e:
    print('except:', e)
finally:
    print('finally...')
print('END')
```
executing the codes using "try", if there is an error, the execution will jump into "except" part instead of executing the rest of codes.

If "finally" part exists, then jump into the "finally" part.

"finally" part will be always executed.
